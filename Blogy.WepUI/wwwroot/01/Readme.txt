## Template Structure ##
> _source: have all template source files and Gulp system for development.
> Default: have the final English/LTR version of template.
> RTL: have the final Arabic/RTL version of template.

## How to edit the template ##
Just open "documentation" foler inside the main template folder and follow the instructions.
